Настроить предобученную GPT-like модель.

В данном домашнем задании предлагается выбрать 
на свое усмотрение одну из открытых генеративных моделей на HuggingFace,
донастроить его под интересующий кейс и реализовать своего бота/агента или даже сеть/цепь агентов.

Имеющиеся мощности:
GPU - GTX 1060 6GB

Идея:
Кулинарный бот — генерирует рецепты по ингредиентам

Модель: GPT-2 small

Датасет: RecipeNLG

Инструменты: Python, transformers, peft (для LoRA), bitsandbytes (для квантизации), datasets — всё ставится через pip.

Результат:
простой веб-интерфейс на Gradio, куда вводишь ингредиенты, а модель генерирует рецепт.
